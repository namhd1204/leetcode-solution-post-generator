746. Min Cost Climbing Stairs Dễ

Trực quan hóa thuật toán Quy hoạch động (Dynamic Programming)

Xem trên LeetCode

Cấu hình bài toán

Mẫu nhanh:
Trực quan hóa
Mục tiêu: Đạt tới đỉnh Top với tổng chi phí ít nhất.
Tổng chi phí hiện tại:
0
Tốc độ:
Bước: 0 / 0

Nhật ký thuật toán ứng với bước hiện tại

Chọn "Áp dụng" hoặc nhấn "Bước tiếp theo" để bắt đầu giải bài toán.

Bảng trạng thái DP

Bậc (i) Cost[i] DP[i] (Min Cost)

Mã nguồn & Dòng đang chạy

const n = cost.length;
// dp[i] là chi phí tối thiểu để đến bậc i
const dp = new Array(n + 1).fill(0);
// Đứng sẵn ở bậc 0 hoặc bậc 1, ko tốn phí
dp[0] = 0;
dp[1] = 0;
for (let i = 2; i <= n; i++) {
dp[i] = Math.min(
dp[i-1] + cost[i-1], // Đi từ i-1 lên 1 bước
dp[i-2] + cost[i-2] // Đi từ i-2 lên 2 bước
);
}
return dp[n];

Mô phỏng trực quan thuật toán giải thích bước đi và cách tính mảng DP.
