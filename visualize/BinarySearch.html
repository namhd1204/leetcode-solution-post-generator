Luồng code đang thực thi

JavaScript
function binarySearch(arr, target) {
    let left = 0;
    let right = arr.length - 1;
    while (left <= right) {
        const mid = Math.floor(left + (right - left) / 2);
        if (arr[mid] === target) {
            return mid; // Tìm thấy!
        }
        if (arr[mid] < target) {
            left = mid + 1; // Thu hẹp sang phải
        } else {
            right = mid - 1; // Thu hẹp sang trái
        }
    }
    return -1; // Không tìm thấy
}

Tìm kiếm Nhị phân (Binary Search) là một thuật toán cực kỳ hiệu quả để tìm kiếm một phần tử trong một danh sách đã được sắp xếp. Thay vì duyệt tuần tự qua từng phần tử (Linear Search) mất thời gian tối đa $O(n)$, Binary Search hoạt động theo phương pháp **"Chia để trị"** (Divide and Conquer).

Thuật toán liên tục chia đôi khoảng tìm kiếm. Tại mỗi bước, nó so sánh giá trị cần tìm kiếm với phần tử ở giữa mảng:

  • Nếu phần tử ở giữa bằng mục tiêu cần tìm, thuật toán kết thúc thành công.
  • Nếu mục tiêu nhỏ hơn phần tử ở giữa, phạm vi tìm kiếm tiếp theo bị thu hẹp về nửa bên trái.
  • Nếu mục tiêu lớn hơn phần tử ở giữa, phạm vi tìm kiếm tiếp theo bị thu hẹp về nửa bên phải.

Độ phức tạp thuật toán (Time Complexity)

Trường hợp tốt nhất (Best Case) O(1) (Tìm thấy ngay giữa)
Trường hợp trung bình (Average Case) O(log n)
Trường hợp tệ nhất (Worst Case) O(log n)

Điều kiện bắt buộc

Mảng đầu vào bắt buộc phải được sắp xếp (tăng dần hoặc giảm dần). Nếu mảng chưa sắp xếp, việc chia đôi sẽ phá vỡ tính quy luật của thuật toán và đưa ra kết quả hoàn toàn sai lệch.
